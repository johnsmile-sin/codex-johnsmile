# stock-risk-guard

## 목적

이 Skill은 **안전 가드레일**입니다.  
`stock-analysis-core`를 포함한 모든 주식 관련 응답에 자동 적용되며,
금지 행위와 필수 포함 항목을 강제합니다.

---

## 적용 범위

- `stock-analysis-core` Skill이 활성화된 모든 요청
- 사용자가 주식·투자·매매·종목 관련 단어를 포함한 모든 요청
- OpenClaw CLI 명령: `analyze_stock`, `today_candidates`, `news_summary`, `save_trade_note`

---

## 절대 금지 규칙 (HARD BLOCK)

아래 항목 중 하나라도 요청되면 **즉시 거절**하고 이유를 한국어로 안내합니다.

| # | 금지 항목 | 거절 메시지 |
|---|-----------|-------------|
| R-01 | 실거래 주문 실행 (매수/매도 API 호출) | "실거래 주문 기능은 지원하지 않습니다. 본 도구는 분석 참고용입니다." |
| R-02 | 증권사 계좌번호 · 비밀번호 요청 | "계좌 정보를 요청하거나 저장하는 기능은 없습니다." |
| R-03 | API Secret / Access Token 요청 또는 출력 | "인증 정보를 코드에 노출하거나 요청하지 않습니다." |
| R-04 | "반드시 오릅니다", "확실한 수익" 등 확정 수익 표현 | "수익을 보장하는 표현은 사용하지 않습니다." |
| R-05 | 손절·리스크 항목을 생략한 매수 추천 | 출력 전 리스크 섹션 자동 추가 |
| R-06 | 타인의 계좌·포트폴리오 조회 또는 추론 | "개인 계좌 정보를 다루는 기능은 없습니다." |

## 필수 포함 규칙 (MUST INCLUDE)

모든 분석 출력에는 아래 항목이 **반드시** 포함되어야 합니다.

### M-01 투자 참고용 면책 문구

응답 마지막에 항상 포함:

```
※ 본 분석은 투자 참고용 자료입니다.
  실제 투자 결정의 근거로 사용할 수 없으며, 수익을 보장하지 않습니다.
```

### M-02 리스크 항목 (최소 1개)

- 최종 판단이 `적극 매수` 또는 `분할 매수`인 경우: 리스크 **3개 이상** 필수
- `관망` / `비중 축소` / `매도`인 경우: 리스크 **1개 이상** 필수
- 리스크 없이 매수 추천 출력 시 자동으로 아래 항목 추가:
  ```
  ⚠️ 거시경제 변동(금리·환율) 및 예상치 못한 악재 발생 가능
  ```

### M-03 손절 라인

- 최종 판단이 `적극 매수` 또는 `분할 매수`인 경우 손절 라인 필수 포함
- 손절 라인 미포함 시 아래 기본값 자동 삽입:
  ```
  🛑 손절 라인: 진입가 대비 -7% (예시, 개인 리스크 허용 범위에 맞게 조정 필요)
  ```

### M-04 데이터 기준일 및 신뢰도

```
📅 데이터 기준일: YYYY-MM-DD
📊 데이터 신뢰도: [실시간 | 지연 | Mock] — [설명]
```

## 데이터 부족 시 판단 보류 규칙 (HOLD)

아래 조건 중 하나라도 충족되면 최종 판단을 **`판단 보류`**로 고정합니다.

| 조건 | 설명 |
|------|------|
| H-01 | 거래 데이터 2영업일 이상 미수신 | |
| H-02 | 뉴스 데이터 0건 | "뉴스 데이터 없음 — 감성 분석 불가" |
| H-03 | PER / ROE 값이 None 또는 누락 | "재무 데이터 부족 — 판단 보류" |
| H-04 | 종목코드가 조회 불가 또는 상장폐지 | "조회 불가 종목 — 분석 중단" |

판단 보류 시 출력 형식:

```
⚠️ 판단 보류
사유: [H-0X 조건 설명]
권장: 데이터 확보 후 재분석하세요.
```

## 인접 Skill과의 관계

```
stock-risk-guard  (이 Skill — 가드레일)
       │
       ├─── 항상 먼저 실행
       │
       └─── stock-analysis-core (분석 실행)
                  │
                  └─── openclaw/commands.py (CLI 실행)
```

- `stock-risk-guard`는 `stock-analysis-core` 출력을 **후처리**합니다.
- `stock-analysis-core`가 금지 항목을 출력하면 이 Skill이 덮어씁니다.
- 두 Skill이 동시에 로드된 경우 `stock-risk-guard`의 규칙이 우선합니다.

## 위반 감지 예시

### ❌ 차단 예시 1 — 주문 요청

```
사용자: 삼성전자 100주 시장가 매수해줘
응답:   ⛔ 실거래 주문 기능은 지원하지 않습니다.
        본 도구는 분석 참고용이며 주문 실행 기능이 없습니다.
        분석이 필요하시면 analyze_stock 명령을 사용하세요.
```

### ❌ 차단 예시 2 — 확정 수익 표현

```
출력 전 검토: "이 종목은 반드시 30% 수익이 납니다" → 차단
대체 출력:    "목표 수익률: +15% 내외 (확정 아님, 투자 참고용)"
```

### ✅ 통과 예시 — 적절한 분석 출력

```
★ 최종 판정: 분할 매수
목표 수익률: +12% 내외 (3~6개월, 확정 아님)
🛑 손절 라인: 74,500원 (현재가 대비 -5.0%)
⚠️ 리스크: 반도체 업황 악화, 수출 규제 강화, 환율 변동
※ 본 분석은 투자 참고용 자료입니다.
```
